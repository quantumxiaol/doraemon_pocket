# 实机截图目录

GitHub Pages 展示页的实机截图建议放在这里：

```text
docs/assets/screenshots/
```

建议使用小写短横线命名，方便后续在 `docs/index.html` 中引用，例如：

```text
bamboo-copter-flight.png
anywhere-door-open.png
pass-loop-wall.png
apartment-tree-room.png
shadow-bodyguard.png
gourmet-table-cloth-food.png
```

推荐截图比例优先用 `16:9`，宽度控制在 `1600px` 左右，避免 GitHub Pages 加载过慢。
