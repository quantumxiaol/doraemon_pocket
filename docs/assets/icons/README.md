# 展示页图标资源

这里存放 GitHub Pages 展示页使用的图标副本。

源文件来自：

```text
src/main/resources/assets/doraemon_pocket/textures/
```

如果后续重画或新增图标，可以同步复制：

```bash
cp src/main/resources/assets/doraemon_pocket/textures/item/*.png docs/assets/icons/item/
cp src/main/resources/assets/doraemon_pocket/textures/block/*.png docs/assets/icons/block/
cp src/main/resources/assets/doraemon_pocket/textures/entity/*.png docs/assets/icons/entity/
```
