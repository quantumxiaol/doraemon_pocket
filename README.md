# Doraemon Pocket

一个适用于 Minecraft Java 版的 Fabric 小道具 Mod，加入哆啦A梦风格的实用道具。

当前版本已实现：

- 竹蜻蜓
- 空气炮
- 休克枪
- 闪避斗篷
- 时光包袱皮
- 适应灯

## 支持版本

- Minecraft Java Edition 1.20.1
- Fabric Loader
- Fabric API 0.92.9+1.20.1 或兼容版本

## 安装

1. 安装 Minecraft 1.20.1 对应的 Fabric Loader。
2. 下载并安装 Fabric API。
3. 将 `doraemon_pocket-*.jar` 放入游戏目录的 `mods` 文件夹。
4. 启动游戏。

`*-sources.jar` 是源码包，不要放进 `mods` 文件夹。

## 竹蜻蜓

竹蜻蜓需要装备到头盔栏。

- 按住跳跃键：上升
- 按住潜行键：下降
- 按住前后左右移动键：按当前视角方向飞行和转向
- 装备期间：降低摔落风险
- 生存模式主动飞行时：消耗耐久

## 空气炮

空气炮需要拿在手上使用。

- 右键：向准星方向发射空气冲击
- 命中锥形范围内的实体：造成少量伤害并击退
- 距离越远：伤害和击退越低
- 草、藤蔓等无碰撞植物：不会阻挡气浪
- 使用后：进入短暂冷却
- 生存模式使用时：消耗耐久

## 休克枪

休克枪需要拿在手上使用。

- 右键：向准星方向发射电击
- 命中实体：施加高等级缓慢、虚弱和负跳跃效果
- 命中怪物：短时间停用 AI，期间不能主动攻击
- 休克目标受到伤害：立刻苏醒
- Boss 级目标：只受到短暂迟缓和虚弱，不会被停用 AI
- 生存模式使用时：消耗耐久

## 闪避斗篷

闪避斗篷可以装备到胸甲栏，也可以拿在主手或副手。

- 受到伤害时：有概率完全闪避本次伤害
- 被投射物命中时：必定闪避，并尝试把投射物弹飞
- 触发闪避时：播放格挡音效并消耗耐久

## 时光包袱皮

时光包袱皮用于修复工具和装备。

- 一只手拿时光包袱皮
- 另一只手拿受损的可修复物品
- 右键：消耗 30 点经验，将受损物品耐久修满
- 创造模式不消耗经验

## 适应灯

适应灯放在背包或手上即可持续生效。

- 获得“环境适应”效果
- 免疫溺水、火焰、岩浆、冰冻、虚空等环境伤害
- 提供夜视和潮涌能量，改善水下视野和呼吸
- 不免疫怪物、玩家、箭矢等实体伤害

## 当前限制

- 当前版本重点验证玩法逻辑。
- 竹蜻蜓模型已接入，后续可继续补扇叶旋转动画。
- 空气炮当前使用原版贴图和原版音效占位。
- 休克枪当前使用原版贴图、音效和粒子占位。
- 闪避斗篷当前已有闪避机制，但披风外观模型暂未制作。
- 时光包袱皮当前使用红蓝配色占位贴图，后续可继续打磨为更像旗帜布面的图标。
- 适应灯当前先处理服务端环境伤害和水下视线；岩浆、细雪迷雾的客户端渲染优化暂未做。
