# Doraemon Pocket

一个适用于 Minecraft Java 版的 Fabric 小道具 Mod，加入哆啦A梦风格的实用道具。

## 道具一览

当前版本已实现：

- 竹蜻蜓
- 空气炮
- 休克枪
- 闪避斗篷
- 时光包袱皮
- 适应灯
- 桃太郎饭团
- 任意门
- 穿透环
- 穿越空间神奇帽
- 石头帽
- 名刀“电光丸”
- 铜锣烧

## 支持版本

- Minecraft Java Edition 1.20.1
- Fabric Loader
- Fabric API 0.92.9+1.20.1 或兼容版本

## 安装

1. 安装 Minecraft 1.20.1 对应的 Fabric Loader。
2. 下载并安装 Fabric API。
3. 将 `doraemon_pocket-*.jar` 放入游戏目录的 `mods` 文件夹。
4. 启动游戏。

`*-sources.jar` 是源码包，不要放进 `mods` 文件夹。

## 竹蜻蜓

竹蜻蜓需要装备到头盔栏。

竹蜻蜓的早期贴图和头戴模型结构参考了 teaconmc 的示例项目 [DoraemonProps](https://github.com/teaconmc/DoraemonProps)。当前 Fabric 1.20.1 版本的装备渲染、飞行逻辑、音效混合和交互行为由本项目重新实现。

- 按住跳跃键：上升
- 按住潜行键：下降
- 按住前后左右移动键：按当前视角方向飞行和转向
- 装备期间：降低摔落风险
- 生存模式主动飞行时：消耗耐久
- 当前拥有 6000 点耐久
- 佩戴后：旋翼会转动，离地飞行时转速更快
- 飞行时：使用原版蜜蜂飞行声和幻翼振翅声混合出竹蜻蜓音效，不额外打包音频素材

## 空气炮

空气炮需要拿在手上使用。

- 右键：向准星方向发射空气冲击
- 命中锥形范围内的实体：造成少量伤害并击退
- 距离越远：伤害和击退越低
- 草、藤蔓等无碰撞植物：不会阻挡气浪
- 使用后：进入短暂冷却
- 生存模式使用时：消耗耐久

## 休克枪

休克枪需要拿在手上使用。

- 右键：向准星方向发射电击
- 命中普通实体：施加“休克”和虚弱效果，短时间内难以移动或攻击
- 命中怪物：持续清除仇恨目标并停止导航
- 休克目标受到伤害：立刻苏醒
- Boss 级目标：只受到短暂迟缓和虚弱，不会被完全休克
- 生存模式使用时：消耗耐久

## 闪避斗篷

闪避斗篷可以装备到胸甲栏，也可以拿在主手或副手。

- 受到伤害时：有概率完全闪避本次伤害
- 普通非环境伤害：当前闪避概率为 32%
- 被投射物命中时：必定闪避，并尝试把投射物弹飞
- 爆炸、岩浆、仙人掌、虚空等环境伤害或没有明确攻击者的伤害：当前不作为闪避斗篷的主要防护对象
- 触发闪避时：播放格挡音效并消耗耐久

## 时光包袱皮

时光包袱皮用于修复工具和装备。

- 一只手拿时光包袱皮
- 另一只手拿受损的可修复物品
- 右键：消耗 30 点经验，将受损物品耐久修满
- 创造模式不消耗经验

## 适应灯

适应灯需要拿在手上照射目标。

- 右键：向准星方向发射绿色适应光
- 命中玩家或生物：赋予 90 分钟“环境适应”效果
- 潜行右键：照射自己
- 环境适应期间：免疫溺水、火焰、岩浆、冰冻、虚空等环境伤害
- 环境适应期间：提供夜视和潮涌能量，改善水下视野和呼吸
- 不免疫怪物、玩家、箭矢等实体伤害

## 桃太郎饭团

桃太郎饭团用于临时安抚生物。

- 右键生物：消耗 1 个饭团，使目标短时间放弃仇恨并听从玩家
- 右键空气：像雪球一样抛出饭团，命中生物后触发同样效果
- 对狼、猫、马等可驯服生物：会直接按原版归属逻辑驯服
- 对普通生物：只提供临时安抚、跟随和骑乘，不会永久改写归属
- 生效期间：目标不会伤害喂食它的玩家，距离适中时会跟随
- 右键大体型目标：可以直接骑乘，类似临时隐形鞍
- 潜行右键同一个目标：提前放归，不额外消耗饭团
- 效果结束后：目标恢复自己的原版行为
- 已属于其他玩家的驯服生物：不会被抢夺

## 任意门

任意门需要拿在手上使用。

- 手持任意门执行 `/doraemon_pocket anywhere_door <x> <y> <z>`：写入目标坐标
- 右键方块：消耗手中的任意门，在点击位置和目标坐标生成一组成对传送门
- 普通右键门体：像原版门一样开关
- 走进任意一端：传送到另一端
- 潜行右键任意一端：关闭并回收这组任意门
- 破坏任意一片传送门：也会同步销毁这一组传送门的两端
- 当前只支持同一维度内连接

## 穿透环

穿透环需要拿在手上对方块表面使用。

- 右键墙面、地面或天花板：沿点击面的内侧方向扫描
- 找到穿过方块后的第一个空腔：消耗手中的穿透环，在两端生成一组成对穿透环
- 普通右键穿透环或穿过任意一端：传送到另一端
- 潜行右键任意一端：回收这组穿透环
- 破坏任意一片穿透环：会同步销毁这一组穿透环的两端
- 如果后面没有空腔：不会生成穿透环

## 穿越空间神奇帽

穿越空间神奇帽需要装备到头盔栏。

- 朝墙面移动：可以穿过普通方块
- 进入方块内部后：获得类似旁观者模式的穿行移动能力
- 空格：向上穿行
- 潜行：向下穿行
- 禁止穿越：基岩、黑曜石、哭泣的黑曜石、强化深板岩、屏障、传送门和末地传送门框架
- 离开方块内部后：恢复普通移动，不会在空气中自由飞天
- 生存模式在方块内部穿行时：消耗耐久

## 石头帽

石头帽需要装备到头盔栏。

- 戴上后：怪物会把玩家当成路边石头一样无视
- 已经锁定玩家的怪物：会被持续清除仇恨
- 怪物近战或怪物投射物：不会伤害戴着石头帽的玩家
- 接触水时：石头帽会自动脱落并回到背包，背包满时掉在地上

## 名刀“电光丸”

名刀“电光丸”需要拿在主手或副手。

- 正面飞来的箭矢、火球、三叉戟等投射物：会被自动弹开
- 身边约 3 格内有生物攻击玩家时：会自动挥剑反击并击退目标
- 身边苦力怕开始引信时：会自动打断并击退
- 接近末影龙身体或龙翼部件时：会自动格挡龙的碰撞/扫击伤害，并把玩家轻微弹开
- 基础攻击较高，满冷却普通攻击约两刀击败无装备僵尸
- 可以像原版剑一样附魔锋利等武器附魔
- 每次自动格挡或反击：消耗较多耐久
- 有电状态当前拥有 640 点自动功能耐久
- 自动功能耐久耗尽时：会变成“没电的电光丸”
- 没电的电光丸：外观相同，不再自动格挡或反击，但仍可作为高伤害剑使用

## 铜锣烧

铜锣烧是一种食物，也预留为后续哆啦A梦 NPC 的交易物。

- 可以直接食用
- 使用小麦、牛奶、可可豆、糖和鸡蛋合成
- 当前合成一次获得 4 个

## 植物改造液

植物改造液用于把普通植物临时改造成奇妙植物。

- 右键原木：在附近树干侧面长出多个椰子果实
- 右键树上的椰子果实：采摘并获得椰子
- 手持椰子右键：打开椰子，获得 1 到 3 个随机原版食物
- 生存模式使用植物改造液后会返还玻璃瓶

## 公寓树

公寓树是独立道具，不通过植物改造液获得。

- 右键地面：种下公寓树苗
- 对公寓树苗使用骨粉：向地下生成一个带梯子的 3x3 或 4x4 竖井、中央大厅、根系式分叉走廊和随机房间
- 地下房间当前使用桦木地板、木墙和简单照明，不生成复杂内饰
- 当前会直接改造地下方块，建议不要在重要建筑正下方催熟

## 天气箱

天气箱是一个可放置的机器方块，需要配合天气卡片使用。

- 右键放置天气箱
- 手持晴天、下雨、雷暴或下雪卡片右键天气箱：插入卡片并修改当前维度天气
- 再次右键天气箱：弹出并取回当前卡片
- 破坏天气箱：会掉落天气箱本体和内部卡片
- 原版没有独立的全局“雪天”，下雪卡会开启降水；寒冷群系会显示为下雪

## 翻译魔芋

翻译魔芋是一种可食用道具。

- 食用后：获得 20 分钟“通晓语言”状态效果
- 状态期间：直视末影人不会激怒它们，附近已经对玩家发怒的末影人也会被安抚
- 状态期间：猪灵会把玩家视为穿着金甲，并持续清除对玩家的仇恨
- 状态期间：与村民交易时，交易价格会被压到最低

## 哆啦A梦的百宝袋

哆啦A梦的百宝袋是一个随身储物道具。

- 右键百宝袋：打开内部背包
- 按 B：打开背包中第一个百宝袋
- 当前拥有 64 个储物格
- 百宝袋不能放进另一个百宝袋
- 内部物品保存在百宝袋自身的物品数据中

## 美食桌布

美食桌布像地毯一样放在地上使用。

- 潜行右键桌布：消耗 1 级经验并随机生成优质食物
- 可能生成牛排、金胡萝卜、南瓜派、熟肉、炖菜等原版食物
- 创造模式点餐不消耗经验
- 普通右键不会触发点餐，避免误操作

## 影子剪刀与影子胶水

影子剪刀可以剪下自己的影子，影子胶水可以把影子粘回去。

- 手持影子剪刀右键：召唤自己的影子
- 影子在前 5 分钟会跟随玩家、捡起附近掉落物并放入玩家背包
- 玩家攻击怪物或被怪物攻击时：影子会帮助攻击目标
- 5 分钟后如果没有用影子胶水收回：影子会变成红眼敌对状态并追杀主人
- 手持影子胶水右键自己的影子：立刻收回影子，造反后也能收回
- 每名玩家同一时间只能拥有一个影子

## 恶魔护照

恶魔护照需要拿在主手或副手生效。

- 生效时：附近被玩家激怒的怪物会持续被赦免并清除仇恨
- 末影人：直视不会激怒，已有仇恨会被清除
- 猪灵：会把玩家视为穿着金甲，开启箱子或挖守卫方块时不会被激怒
- 铁傀儡、僵尸猪灵、循声守卫等已锁定玩家的目标：会被持续清除
- 当前不提供饰品栏支持，需要实际拿在手上

## 云朵凝固瓦斯

云朵凝固瓦斯需要在高空使用。

- 使用条件：玩家需要在 Y=192 以上
- 右键空气：在玩家视线前方读取坐标噪声，生成一片凝固云岛
- 云岛大小：从约 3x3 的小云到接近 64x64 的大云都有可能出现
- 云岛形状：不规则轮廓，带 1 到 4 格厚度和简单起伏
- 凝固云朵：可以站立、可以建造，落在上面不会受到摔落伤害，并会轻微弹起
- 生存模式使用时：消耗瓦斯耐久
- 生成时：只填充空气，不覆盖已有方块

## 深海面霜

深海面霜是一种可消耗的生存辅助道具。

- 右键涂抹：获得 20 分钟水下呼吸、夜视和海豚的恩惠
- 适合深海探索、海底遗迹和水下建造
- 生存模式使用后：消耗 1 个深海面霜

## 游泳霜

游泳霜是一种可消耗的地下探索道具。

- 右键涂抹：获得 5 分钟“土地潜游”状态效果
- 朝天然泥土、砂石、岩石和矿石移动：可以钻入并穿行
- 进入天然地形内部后：获得类似旁观者模式的上下左右移动能力
- 空格：向上游
- 潜行：向下游，也可以从地表向下潜入脚下天然地形
- 不能穿越：人工方块、木头、玻璃、机器方块、基岩、黑曜石等非允许方块
- 效果结束后：如果还在方块中，会按原版规则开始窒息

## 快速发条

快速发条可以加快生物和机器的行动。

- 右键生物：赋予 1 分钟速度 III 和急迫 III
- 右键熔炉、高炉或烟熏炉：推进约 200 tick 烧炼进度
- 生存模式使用时：消耗耐久

## 樵夫之泉

樵夫之泉是一个可放置方块。

- 把受损旧工具或盔甲丢进泉里：泉水会吞掉旧物并喷出奖励
- 受损铁工具或铁盔甲：当前会奖励对应的钻石工具或钻石盔甲
- 受损木制、石制、金制工具，以及受损皮革、锁链、金制盔甲：当前会奖励对应的铁制品
- 当前先实现核心奖励循环，点击选择“金的/银的/旧的”问答后续再做

## 露营胶囊

露营胶囊需要对地面使用。

- 右键地面：生成 10 到 15 格高的脚手架柱
- 顶部生成一个白色胶囊房间，带粉色底部、玻璃窗、床、炼药锅、灯笼和照明
- 生成前会检查上方空间，空间不足时不会展开
- 生存模式使用后：消耗 1 个露营胶囊

## 还原光线

还原光线需要照射掉落物使用。

- 右键：照向准星前方的掉落物
- 对可合成物品：读取合成配方并掉落一组原材料
- 对熟肉、烤土豆、木炭等少数非合成物：使用内置还原规则
- 生存模式使用时：消耗耐久

## 哆啦A梦与迷你哆啦

哆啦A梦和迷你哆啦当前是第一版生物实体。

- 创造模式物品栏：可以拿到哆啦A梦刷怪蛋和迷你哆啦刷怪蛋
- 命令测试：`/summon doraemon_pocket:doraemon` 或 `/summon doraemon_pocket:mini_doraemon`
- 外观：哆啦A梦使用蓝白主体；迷你哆啦随机红、黄、绿三种主体色；模型包含红鼻子、胡须、口袋、铃铛和头顶竹蜻蜓
- 行为：会悬浮和缓慢飞行，不会自然消失
- 哆啦A梦交易：手持铜锣烧右键，会按手中铜锣烧数量从可负担道具池随机换一个道具
- 迷你哆啦互动：手持铜锣烧右键，会吃掉 1 个铜锣烧并冒爱心
- 当前还不是正式村民式交易 GUI，后续可以改成明确的道具选择界面

## 当前限制

- 当前版本重点验证玩法逻辑。
- 已做基础稳定性处理：临时状态不会写入普通生物归属，常驻扫描已做分频，竹蜻蜓控制包会超时清理。
- 竹蜻蜓模型、基础旋翼动画和原版音效混合已接入，后续可继续微调音量、节奏和平滑转速。
- 空气炮当前已有自制像素图标，音效仍使用原版音效占位。
- 休克枪当前已有自制像素图标，音效和粒子仍使用原版效果占位。
- 闪避斗篷当前已有闪避机制和基础披风外观，披风飘动动画暂未制作。
- 时光包袱皮当前使用红蓝配色占位贴图，后续可继续打磨为更像旗帜布面的图标。
- 适应灯当前是射线命中的绿色照射枪；环境伤害免疫已处理，岩浆、细雪迷雾的客户端渲染优化暂未做。
- 桃太郎饭团当前是临时安抚/跟随机制，不提供宠物坐下交互。
- 任意门当前不提供坐标输入 GUI；先用命令把目标坐标写入手里的任意门，放置后会消耗手中的任意门，且只支持同维度传送。
- 穿透环当前按直线扫描第一个空腔，最大探测距离 96 格，不做复杂洞口尺寸匹配。
- 穿越空间神奇帽当前通过碰撞放行实现穿墙，只阻挡一组硬禁止方块，不做权限、领地或防作弊兼容。
- 名刀“电光丸”的自动反击当前处理真正打到玩家的近身攻击、正面投射物、已经开始引信的近身苦力怕，以及末影龙的近身碰撞/扫击；不会主动追击其他没有出手的怪物。
- 没电的电光丸当前不可重新充电，后续可以再加充电配方或能源机制。
- 铜锣烧当前先作为食物和后续交易货币预留，NPC 交易尚未制作。
- 植物改造液当前只支持原木，用于长出可采摘椰子果实。
- 公寓树当前会直接改造地下方块，建议不要在重要建筑正下方催熟。
- 天气箱的下雪卡遵循原版天气限制：本质是开启降水，只有寒冷群系会显示雪。
- 翻译魔芋当前只处理末影人、猪灵和村民交易价格，不会翻译聊天内容。
- 百宝袋当前使用自定义 64 格界面，不支持把百宝袋套进百宝袋。
- 美食桌布当前是随机点餐，不提供输入菜名的 GUI。
- 影子当前是黑色人形随从，不复制玩家皮肤、装备或完整属性。
- 恶魔护照当前通过主手或副手触发，不接入饰品栏；部分其他 Mod 自定义仇恨系统可能不受影响。
- 云朵凝固瓦斯当前只在 Y=192 以上生成云岛；云形是确定性坐标噪声，不做真实天气云层同步。
- 游泳霜当前只允许穿行一组天然地形方块，黑暗视野通过隐藏卡墙遮罩和夜视缓解，但没有专门的地下透视渲染。
- 樵夫之泉当前是自动奖励 MVP，不提供点击问答和撒谎惩罚。
- 露营胶囊当前直接生成简单结构，不保存“胶囊所有者”，也不支持一键回收。
- 还原光线当前优先支持原版合成配方和少量手写还原规则，不做复杂递归配方树。
- 哆啦A梦当前通过刷怪蛋或 summon 命令生成，不做自然生成；交易是随机道具池，不是正式交易 GUI。
- 迷你哆啦当前是红/黄/绿随机变体的陪伴型小实体，只吃铜锣烧和飞行，不提供交易。
- 击败末影龙后会在末地播放一次哆啦A梦竹蜻蜓飞行动画；测试命令为 `/doraemon_pocket doraemon_flyby`。
