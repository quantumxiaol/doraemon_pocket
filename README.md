# Doraemon Pocket

一个适用于 Minecraft Java 版的 Fabric 小道具 Mod，加入哆啦A梦风格的实用道具。

当前版本先实现了第一个道具：竹蜻蜓。

## 支持版本

- Minecraft Java Edition 1.20.1
- Fabric Loader
- Fabric API 0.92.9+1.20.1 或兼容版本

## 安装

1. 安装 Minecraft 1.20.1 对应的 Fabric Loader。
2. 下载并安装 Fabric API。
3. 将 `doraemon_pocket-*.jar` 放入游戏目录的 `mods` 文件夹。
4. 启动游戏。

`*-sources.jar` 是源码包，不要放进 `mods` 文件夹。

## 竹蜻蜓

竹蜻蜓需要装备到头盔栏。

- 按住跳跃键：上升
- 按住潜行键：下降
- 按住前后左右移动键：按当前视角方向飞行和转向
- 装备期间：降低摔落风险
- 生存模式主动飞行时：消耗耐久

## 当前限制

- 当前版本重点验证玩法逻辑。
- 竹蜻蜓模型已接入，后续可继续补扇叶旋转动画和更多道具。
